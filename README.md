# ⚽ Simulador de Copa do Mundo - Full Stack Challenge

## 📌 Sobre o projeto

Este projeto é um simulador completo de uma Copa do Mundo, desenvolvido como parte de um desafio Full Stack.  
A aplicação realiza a geração de grupos, simulação de partidas, fase de mata-mata com pênaltis e envio do resultado final para uma API.

O objetivo do projeto é aplicar conceitos de desenvolvimento web, manipulação de dados, consumo de API e lógica de programação.

---

## 🚀 Funcionalidades

- 🔄 Consumo de API para obtenção de dados  
- 🏆 Geração automática dos grupos da Copa  
- ⚽ Simulação de fase de grupos  
- 📊 Cálculo de tabela e classificação  
- 🥊 Fase eliminatória (mata-mata)  
- 🎯 Disputa de pênaltis em caso de empate  
- 📡 Envio do resultado final para API externa  
- 🎮 Interface interativa para simulação
